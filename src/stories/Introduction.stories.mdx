# React-ngm-form

Is a React component library for build forms with validations using [**React-form-hook**](https://react-hook-form.com/) and [**Reactstrap**](https://reactstrap.github.io/) with [**Bootstrap 5.1**](https://getbootstrap.com/docs/5.1/getting-started/introduction/)


<div className="subheading">Instalation</div>

<div>
  <Source
    format={false}
    code={`npm i --save react-ngm-form`}
  />
</div>
